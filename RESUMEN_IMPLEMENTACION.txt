┌─ FLUJO DE AUTENTICACIÓN (OAuth2) ───────────────────────────────────────────┐
│                                                                              │
│  Usuario A                            Servidor                Microsoft     │
│  ─────────────────────────────────────────────────────────────────────     │
│                                                                              │
│  1. Hace click en                                                           │
│     "Conectar con Microsoft"                                               │
│           │                                                                │
│           ├─► GET /api/auth/microsoft/login ─────────────────────────►   │
│           │                                  Devuelve URL                 │
│           │◄────────────────────────────────────────────────────────────── │
│           │                                                                │
│  2. Abre URL de Microsoft                                                 │
│  3. Inicia sesión con cuenta Microsoft  (login.microsoftonline.com)     │
│  4. Aprueba permisos (Mail.Read, User.Read, offline_access)            │
│  5. Microsoft redirige con código                                        │
│           │                                                                │
│           ├─► GET /api/auth/callback/microsoft?code=xxxxx ──────────────►│
│           │                   Backend intercambia código por tokens       │
│           │                   Crea usuario + guarda tokens                │
│           │◄──────────────────────────────────────────────────────────────│
│           │                   Devuelve JWT tokens                        │
│           │                                                                │
│  6. Guarda JWT token en localStorage                                     │
│                                                                              │
│  7. Puede sincronizar:                                                     │
│     GET /api/sync/microsoft/sent ─────────────────────────────────►       │
│     (Los correos SOLO de ana@ se guardan en la BD)                       │
│                                                                              │
└────────────────────────────────────────────────────────────────────────────────┘

┌─ DATOS ALMACENADOS POR USUARIO ─────────────────────────────────────────────┐
│                                                                              │
│ Modelo User (actualizado):                                                 │
│ ├─ microsoft_id          (ID único de Microsoft)                          │
│ ├─ microsoft_email       (Email de Office 365)                            │
│ ├─ access_token          (Token para Graph API)                           │
│ ├─ refresh_token         (Para renovar access_token)                      │
│ └─ token_expires_at      (Fecha de expiración)                            │
│                                                                              │
│ Aislamiento:                                                               │
│ • Email.query.filter_by(user_id=usuario.id) → Solo SUS correos          │
│ • Cada usuario tiene su propio access_token → Su propia caja de correos  │
│ • Microsoft autentica la identidad → Garantía de seguridad               │
│                                                                              │

┌─ PRÓXIMOS PASOS ────────────────────────────────────────────────────────────┐
│                                                                              │
│ Para poner en producción:                                                  │
│                                                                              │
│ 1️⃣  CONFIGURAR AZURE AD (15 minutos)                                      │
│    • Va a https://portal.azure.com                                        │
│    • Crea nueva aplicación en "App registrations"                         │
│    • Agrega permisos: Mail.Read, User.Read, offline_access              │
│    • Genera Client Secret                                                 │
│    • Copia: Client ID, Secret, Tenant ID                                 │
│                                                                              │
│ 2️⃣  CREAR .env CON CREDENCIALES (2 minutos)                              │
│    MICROSOFT_CLIENT_ID=xxxx                                               │
│    MICROSOFT_CLIENT_SECRET=xxxx                                           │
│    MICROSOFT_TENANT_ID=xxxx                                               │
│    MICROSOFT_REDIRECT_URI=http://localhost:5000/api/auth/callback/...    │
│                                                                              │
│ 3️⃣  INSTALAR DEPENDENCIAS (1 minuto)                                      │
│    pip install -r requirements.txt                                        │
│                                                                              │
│ 4️⃣  CORRER APLICACIÓN (1 minuto)                                          │
│    python run.py                                                          │
│                                                                              │
│ 5️⃣  PROBAR (5 minutos)                                                    │
│    • Navega a http://localhost:5000                                       │
│    • Conecta con Microsoft                                                │
│    • Sincroniza correos                                                   │
│    • Verifica que se guardaron correctamente                              │
│                                                                              │
│ Total: ~25 minutos ⏱️                                                       │
